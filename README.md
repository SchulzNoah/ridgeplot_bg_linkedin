## Ridge Plot: Left-Right Self-Placement by Party Vote

This repository contains a concise R script for visualizing the left-right self-placement by party vote. The analysis is based on the GLES 2025 (ZA10100) dataset. Due to licensing restrictions, the raw `.dta` file is not included. It can be obtained via [GESIS - Leibniz Institute for the Social Sciences](https://search.gesis.org/research_data/ZA10100).
